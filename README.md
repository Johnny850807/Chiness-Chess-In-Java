# Chiness-Chess-In-Java
How to develop a Chiness Chess Game in ONE DAY in JAVA JFrame ?

The project shows an answer.

## To launch the game 

1. go to the command window.
2. type `cd ChinessChessGame`
3. type `java -jar ChinessChessGame.jar`
